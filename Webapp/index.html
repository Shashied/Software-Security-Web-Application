What is OAuth?

OAuth definition OAuth is an open-standard authorization protocol or framework that describes how unrelated servers and services can safely allow authenticated access to their assets without actually sharing the initial, related, single logon credential

OAuth 2.0 Work Flow

  • Your app requests an authorization from a user with a button
  • Once granted, your app receives a temporary code.
  • Call the oauth.access method with the code/li>
  • Exchange the code to a more permanent token
  • Use the token to make API calls
  • Slack server gives your app responses
